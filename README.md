# GLOWdex (Frontend)

Modern frontend rebuild for the Global Coastal Wetlands Index application.

## Prerequisites

- Node.js (v18+)
- npm (v9+)

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Stack

- **Vite** - Build tool
- **React** - UI framework
- **TypeScript** - Language
- **Tailwind CSS** - Styling
